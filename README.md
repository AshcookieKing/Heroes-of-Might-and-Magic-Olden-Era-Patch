## Установка

1. Закройте игру.
2. Запустите установщик.
3. Укажите папку с `HeroesOldenEra.exe` и `GameAssembly.dll`.
4. «Далее» → «Установить» → «Закрыть».

После обновления Steam — снова тот же exe.

---

## Мультиплеер

Патч **у каждого** игрока в лобби. Хост + гости. Иначе будет рассинхрон.

---

## Откат

Пуск → **SimTurnsAI Patch** → **Otkat patcha**
